# 欢迎阅读

## 概述

在工作过程中翻译的一部分英文资料的中文版，整理出来以静态页面的方式放在这里方便大家查阅。

几年前花费了一个月翻译的JDBC标准文档丢失了很伤心。与其丢失不如分享出来供有需要的人共同使用。如果认为分享的资料有价值请给我**加星或留言**，也可以把您的想法通过issues告诉我。^_^
